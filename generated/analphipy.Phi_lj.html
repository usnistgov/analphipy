analphipy.Phi_lj

class analphipy.Phi_lj(sig=1.0, eps=1.0)[source]

Lennard-Jones potential.

\[\phi(r) = 4 \epsilon \left[ \left(\frac{{\sigma}}{{r}}\right)^{{12}} - \left(\frac{{\sigma}}{{r}}\right)^6\right]\]
Parameters

  • r_min (float, optional) – Position of minimum.

  • phi_min (float, optional) – Value of phi at r_min.

  • segments (sequence of ints) – Integration limits. For n = len(segmetns) integration will be performed over ranges (segments[0], segments[1]), (segments[1], segments[2]), ..., (segments[n-2], segments[n-]])

  • sig (float) – Length parameter \(\sigma\).

  • eps (float) – Energy parameter \(\epsilon\).

Notes

Specific subclasses should set values for r_min, phi_min, and segments, as well as forms for phi and dphidr.
