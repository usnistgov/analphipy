analphipy.Phi_sw.minimize

Phi_sw.minimize(r0, bounds=None, **kws)[source]

Determine position r where phi is minimized.

Parameters

  • r0 (float) – Guess for position of minimum. If value is ‘mean’, then use r0 = mean(bounds).

  • bounds (tuple floats, {'segments'}, optional) – Bounds for minimization search. If tuple, should be of form bounds=(lower_bound, upper_bound). If ‘segments, then bounds=(segments[0], segments[-1]). If None, no bounds used.

  • **kws – Extra arguments to analphipy.minimize_phi()

Returns

  • rmin (float) – phi(rmin) is found location of minimum.

  • phimin (float) – Value of phi(rmin), i.e., the value of phi at the minimum

  • output – Output class from scipy.optimize.minimize().
