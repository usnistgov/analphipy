analphipy.utils.quad_segments(func, segments, args=(), full_output=False, sum_integrals=True, sum_errors=False, err=True, **kws)[source]ÔÉÅ

Perform quadrature with discontinuities

Parameters

  • func (callable()) ‚Äì function to be integrated

  • segments (sequence of ints) ‚Äì Integration limits. For n = len(segmetns) integration will be performed over ranges (segments[0], segments[1]), (segments[1], segments[2]), ..., (segments[n-2], segments[n-]])

  • args (tuple, optional) ‚Äì Extra positional arguments to func.

  • full_output (bool, default False.) ‚Äì If True, return extra information.

  • sum_integrals (bool, default True) ‚Äì If True, sum the segments in the output.

  • sum_errors (bool, default True) ‚Äì If True and returning error sum errors.

  • err (bool, default = True) ‚Äì If True, return error.

  • **kws ‚Äì Extra arguments to scipy.integrate.quad()

Returns

  • integral (float or list of floats) ‚Äì If sum_integrals, this is the sum of integrals over each segment. Otherwise return list of values corresponding to integral in each segment.

  • errors, float or list of floats, optional ‚Äì If err or full_output are True, then return error. If sum_errors, then sum of errors Across segments.

  • outputs ‚Äì Output from scipy.integrate.quad(). If multiple segments, return a list of output.
